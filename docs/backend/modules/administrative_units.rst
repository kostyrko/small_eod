Jednostki podziału terytorialnego
=================================

Założenia
---------

Moduł odpowiedzialny za dostarczanie informacji o jednostkach podziału terytorialnego w Polsce.
Zapewnia przegląd instytucji w danym regionie. Oparty jest o moduł `django-teryt-tree <https://github.com/ad-m/django-teryt-tree>`_ dla którego istnieje odrębna dokumentacja.
